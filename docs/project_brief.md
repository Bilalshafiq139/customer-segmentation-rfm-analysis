# Project Brief

## Project Title

RFM Customer Segmentation Analysis

## Problem Statement

The business needs a practical way to identify valuable customers, retain loyal buyers, reactivate inactive customers, and improve campaign targeting. Transaction-level retail data can be converted into customer-level behavioral segments using RFM analysis.

## Objective

Build a customer segmentation analysis using Recency, Frequency, and Monetary value, then convert the results into clear marketing recommendations.

## Methodology

- Cleaned and prepared online retail transaction data.
- Converted invoice dates into a usable time-based format.
- Calculated customer-level Recency, Frequency, and Monetary metrics.
- Created RFM scores and assigned customers to business-friendly segments.
- Compared segments by purchase behavior, geography, and product patterns.
- Built supporting dashboard, report, and presentation outputs.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

## Business Value

The analysis helps marketing and sales teams decide which customers should receive loyalty offers, reactivation campaigns, premium promotions, and product-focused recommendations.

## Business Deliverables

- End-to-end analytics workflow from raw transaction data to business recommendations.
- Customer segmentation using a widely used marketing analytics method.
- Python analysis and Power BI dashboarding.
- Findings communicated through notebook, dashboard, report, and presentation formats.
- Technical analysis connected to practical business action.
